# Linguagem C - Fundamentos da Programação

Repositório dedicado aos exercícios e desafios desenvolvidos na disciplina de Linguagem C.

O objetivo deste repositório é consolidar a base em lógica de programação e fundamentos essenciais para a formação em Engenharia de Software.

---

## Estrutura do Projeto

super_trunfo/
 ├── novato/
 ├── intermediario/
 └── mestre/

Cada nível trabalha conceitos progressivos de lógica e estrutura.

---

## Conceitos praticados

- Variáveis e tipos primitivos  
- Entrada e saída de dados  
- Estruturas condicionais (if, else)  
- Operadores aritméticos e relacionais  
- Cálculo de indicadores (densidade populacional, PIB per capita, etc.)  
- Organização e clareza no código  

---

## Objetivo

Fortalecer a base em programação estruturada, desenvolvendo raciocínio lógico e boas práticas de organização de código.
